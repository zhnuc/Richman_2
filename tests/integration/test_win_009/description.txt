测试用例：test_win_009
功能模块：胜负判定
测试目标：验证多人游戏中玩家依次破产的处理

测试描述：
1. 使用preset加载三人游戏中Q和A资金都很少（各100元）的状态
2. Q先移动到S的1级房产（过路费200元）破产，游戏继续
3. A再移动到S的1级房产（过路费200元）也破产
4. 验证S作为最后一人获胜，游戏结束

验证内容：
- Q先破产，alive状态变为false，游戏继续
- A再破产，alive状态变为false
- S作为最后存活者获胜
- 游戏状态设为ended=true，winner设为S的index（2）

测试重点：
- 验证连续破产的处理机制
- 确保破产顺序的正确处理
- 验证最终获胜条件的判定
